Sass Compiler & Dev Server Starter Pack!

This small boilerplate allows you to get started quickly for compiling Sass to CSS. Tools & technologies used are:

  • Gulp.js
  • Gulp-Sass
  • Browser Sync
  • Bootstrap 4 Beta
  • Font Awesome

All Sass code is inside the style.scss file (located in the sccs folder).

Gulp.js will watch any changes to the style.scss file and changes to any HTML files. Browser Sync will automatically reload the webpage every time there is a save.
